 Create filter and template steps – In progress (draft below)

1. Upload into source folder

2. Loop through source files

3. Create/open filter

4. Display first file in right panel of split screen

5. Double-click word/select words to be searched/replaced (in right panel of split screen)

6. Drag-drop word from right panel, Into left panel. Select tag for the word/create new tag

7. Add selected word and tag into open filter

8. Replace all occurrences of the word with the new tag (throughout the open file)

9. Repeat steps 5-8, until all desired words have been replaced with tags (throughout the open file)

10. Save the file (with the newly added tags), with the “Save As”, new template (per naming convention)

11. Refresh file display in the right panel

12. Display next file in right panel of split screen

13. Repeat steps 9-12

14. After the last source file has been processed and the template file has been saved — sort the tags in the filter file; remove duplicates.

15. Save the filer file per naming standards/update the database
